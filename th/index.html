สมุดฝึกหัด Python

    เขียน Python บนเบราว์เซอร์ได้เลย!
    สมุดฝึกหัดเขียนโปรแกรมสำหรับเด็กๆ

    ไม่ต้องติดตั้ง ข้อผิดพลาดเป็นภาษาไทย เด็กๆ ก็เข้าใจง่าย

    ลองเลย

    เคยเจอแบบนี้ไหม?

    • 😢 พยายามติดตั้ง Python แต่ยากเกินไปก็เลยเลิก
    • 💻 ติดตั้งโปรแกรมบน Chromebook ไม่ได้
    • 🌍 เครื่องมือ Python ออนไลน์เป็นภาษาอังกฤษหมด — ยากเกินไปสำหรับเด็กๆ
    • ❓ ข้อความแจ้งข้อผิดพลาดเป็นภาษาอังกฤษ — ไม่รู้ต้องแก้อะไร

    คุณสมบัติ

    🌐

    ไม่ต้องติดตั้ง

    แค่เปิดเบราว์เซอร์ ใช้ได้บน Chromebook และแท็บเล็ตด้วย

    ✏️

    เป็นภาษาไทย

    ปุ่มและเมนูเป็นภาษาไทย เด็กๆ ใช้ง่าย

    💬

    ข้อผิดพลาดเข้าใจง่าย

    แปลข้อผิดพลาดภาษาอังกฤษเป็นคำที่เด็กๆ เข้าใจ บอกวิธีแก้ด้วยนะ

    💾

    โค้ดไม่หาย

    บันทึกอัตโนมัติ ปิดเบราว์เซอร์แล้วโค้ดก็ยังอยู่

    ถ้ามีข้อผิดพลาดล่ะ?

    ข้อความแจ้งข้อผิดพลาดภาษาอังกฤษจะถูกแปลเป็นคำที่เด็กๆ เข้าใจ บอกวิธีแก้ด้วยนะ

    ข้อผิดพลาดภาษาอังกฤษ SyntaxError: '(' was never closed
    ↓
    แปลแล้ว วงเล็บ ( ยังไม่ได้ปิดนะ ใส่ ) ด้วย
    ข้อผิดพลาดภาษาอังกฤษ NameError: name 'hensuu' is not defined
    ↓
    แปลแล้ว hensuu คืออะไร? ยังไม่ได้สร้าง หรืออาจสะกดผิดนะ
    ข้อผิดพลาดภาษาอังกฤษ IndentationError: expected an indented block
    ↓
    แปลแล้ว เว้นวรรคไม่พอนะ หลัง if หรือ for ต้องเว้น 4 ช่อง

    รองรับข้อผิดพลาดมากกว่า 20 ชนิด

    หลัง Scratch แล้วเรียนอะไรต่อ?

    เรียนรู้การสร้างโปรแกรมด้วยการต่อบล็อกใน Scratch ได้แล้ว เก่งมาก!

    แต่ถ้าอยากเขียนโปรแกรมด้วยตัวอักษร ก้าวต่อไปคือ Python

    ด้วยสมุดฝึกหัด Python เขียนบล็อก "ทำซ้ำ" และ "ถ้า" เหมือนใน Scratch ได้ แต่เป็น Python

    ส่วน "ตัวอย่าง" ในแอปมีโค้ด Python ที่ทำสิ่งเดียวกับบล็อก Scratch

    ดูตัวอย่าง

    ดูสิว่าทำอะไรได้บ้าง!

    🎨 สร้างลายเป็นรูปแบบ

    for i in range(5):
    print("★" * i)
    ★ ★★ ★★★ ★★★★

    🎵 สร้างเพลง

    for not_phleng in ["โด","เร","มี","ฟา","ซอล"]:
    print(not_phleng, "♪")
    โด ♪ เร ♪ มี ♪ ...

    🚀 ปล่อยจรวด

    for i in [3,2,1]:
    print(i)
print("ปล่อย!")
    3 2 1 ปล่อย!
