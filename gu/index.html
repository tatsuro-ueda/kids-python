    ઇન્સ્ટોલ કરવાની જરૂર નથી. ગુજરાતીમાં ભૂલ સંદેશા. તમારી ભાષામાં.

    અત્યારે જ અજમાવો

    આ ઓળખીતું લાગે છે?

    • 😢 Python ઇન્સ્ટોલ કરવા પ્રયાસ કર્યો, ઘણું કઠિન હતું, છોડી દીધું
    • 💻 Chromebook છે એટલે સોફ્ટવેર ઇન્સ્ટોલ નથી થતું
    • 🌍 ઓનલાઇન Python ટૂલ્સ બધા અંગ્રેજીમાં — બાળકો માટે કઠિન
    • ❓ ભૂલ સંદેશા અંગ્રેજીમાં — શું સુધારવું તે ખબર નથી પડતી

    બટન અને મેનૂ ગુજરાતીમાં. નાના વિદ્યાર્થીઓ માટે સરળ.

    💬

    બાળકોને સમજાય એવી ભૂલો

    અંગ્રેજી ભૂલો બાળકોને સમજાય એવા શબ્દોમાં ભાષાંતર થાય છે. કેવી રીતે સુધારવું તે પણ બતાવે છે.

    Scratch પછી શું?

    Scratch માં બ્લોક જોડીને પ્રોગ્રામ બનાવતા શીખ્યા. શાબાશ!

    પણ લખાણમાં પ્રોગ્રામ લખવો હોય તો આગળનું પગલું Python છે.

    Python અભ્યાસ ચોપડીમાં, Scratch માં શીખેલું "ફરી કરો" અને "જો" — Python માં લખી શકો છો.

    એપના "ઉદાહરણો" ભાગમાં Scratch બ્લોક જેવું જ કામ કરતો Python કોડ છે.
