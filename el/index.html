    Χαρακτηριστικά

    🌐

    Χωρίς εγκατάσταση

    Απλά άνοιξε τον browser. Λειτουργεί και σε Chromebook και tablet.

    ✏️

    Όλα στα ελληνικά

    Κουμπιά και μενού στα ελληνικά. Εύκολο ακόμα και για τα μικρότερα παιδιά.

    💬

    Φιλικά σφάλματα

    Τα αγγλικά σφάλματα μεταφράζονται σε λέξεις που τα παιδιά καταλαβαίνουν. Δείχνει και πώς να τα διορθώσεις.

    💾

    Ο κώδικας δεν χάνεται

    Αυτόματη αποθήκευση. Κλείσε τον browser και ο κώδικάς σου θα είναι ακόμα εκεί.

    Κι αν γίνει σφάλμα;

    Τα αγγλικά μηνύματα σφαλμάτων μεταφράζονται σε λέξεις που τα παιδιά καταλαβαίνουν. Δείχνει και πώς να τα διορθώσεις.

    Αγγλικό σφάλμα SyntaxError: '(' was never closed
    ↓
    Μεταφρασμένο Η παρένθεση ( δεν έκλεισε ποτέ. Πρόσθεσε ) για να την κλείσεις
    Αγγλικό σφάλμα NameError: name 'hensuu' is not defined
    ↓
    Μεταφρασμένο Τι είναι το hensuu; Δεν το έχεις δημιουργήσει ακόμα, ή έλεγξε την ορθογραφία
    Αγγλικό σφάλμα IndentationError: expected an indented block
    ↓
    Μεταφρασμένο Δεν φτάνουν τα κενά. Πρόσθεσε 4 κενά μετά το if ή for

    Υποστηρίζει πάνω από 20 διαφορετικούς τύπους σφαλμάτων

    Τι έρχεται μετά το Scratch;

    Έμαθες να φτιάχνεις προγράμματα βάζοντας μπλοκ μαζί στο Scratch. Μπράβο!

    Αλλά αν θέλεις να γράφεις προγράμματα με κείμενο, το επόμενο βήμα είναι η Python.

    Με το Τετράδιο Python μπορείς να γράψεις τα ίδια μπλοκ "επανέλαβε" και "αν" του Scratch — αλλά σε Python.

    Στα "Παραδείγματα" υπάρχει κώδικας Python που κάνει τα ίδια πράγματα με τα μπλοκ του Scratch.

    Δες τα παραδείγματα

    Δες τι μπορείς να κάνεις!
