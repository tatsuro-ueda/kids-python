    ¿Y si hay un error?

    Los mensajes de error en inglés se traducen a palabras que los niños pueden entender. También muestra cómo corregirlos.

    Error en inglés SyntaxError: '(' was never closed
    ↓
    Traducido El paréntesis ( no se cerró. Agrega ) para cerrarlo
    Error en inglés NameError: name 'hensuu' is not defined
    ↓
    Traducido ¿Qué es hensuu? Aún no lo has creado, o revisa el nombre
    Error en inglés IndentationError: expected an indented block
    ↓
    Traducido Faltan espacios. Agrega 4 espacios después de if o for

    Soporta más de 20 tipos de errores diferentes

    ¿Qué viene después de Scratch?

    Aprendiste a hacer programas juntando bloques en Scratch. ¡Increíble!

    Pero si quieres escribir programas con texto, el siguiente paso es Python.

    Con el Cuaderno de Python, puedes escribir los mismos bloques de "repetir" y "si" de Scratch — pero en Python.

    La sección de "Ejemplos" tiene código Python que hace lo mismo que los bloques de Scratch.
