Scratch এর পর কী?

Scratch-এ ব্লক জুড়ে প্রোগ্রাম বানাতে শিখেছো। দারুণ!

কিন্তু লেখা দিয়ে প্রোগ্রাম লিখতে চাইলে, পরের ধাপ হলো Python।

Python অভ্যাস খাতায় Scratch-এর "পুনরাবৃত্তি" আর "যদি" ব্লকগুলো Python-এ লিখতে পারবে।

অ্যাপের "উদাহরণ" অংশে Scratch ব্লকের মতো কাজ করা Python কোড আছে।
