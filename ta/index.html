Python பயிற்சிப் புத்தகம்

    உலாவியிலேயே Python இயங்கும்.
    குழந்தைகளுக்கான நிரலாக்கப் பயிற்சிப் புத்தகம்

    நிறுவ வேண்டாம். தமிழில் பிழைச் செய்திகள். உன் மொழியில்.

    இப்போதே முயற்சி செய்

    இது தெரிந்ததா?

    • 😢 Python நிறுவ முயன்றேன், மிகக் கடினமாக இருந்தது, விட்டுவிட்டேன்
    • 💻 Chromebook-இல் மென்பொருள் நிறுவ முடியாது
    • 🌍 ஆன்லைன் Python கருவிகள் எல்லாம் ஆங்கிலத்தில் — குழந்தைகளுக்கு கடினம்
    • ❓ பிழைச் செய்திகள் ஆங்கிலத்தில் — எதைச் சரி செய்வது என்று தெரியவில்லை

    அம்சங்கள்

    🌐

    நிறுவ வேண்டாம்

    உலாவியைத் திறந்தால் போதும். Chromebook மற்றும் டேப்ளெட்டிலும் வேலை செய்யும்.

    ✏️

    தமிழில்

    பொத்தான்களும் மெனுக்களும் தமிழில். இளம் மாணவர்களுக்கு எளிது.

    💬

    குழந்தைகளுக்கு ஏற்ற பிழைகள்

    ஆங்கிலப் பிழைகள் குழந்தைகளுக்குப் புரியும் வார்த்தைகளில் மொழிபெயர்க்கப்படும். சரி செய்வது எப்படி என்றும் காட்டும்.

    💾

    குறியீடு தொலையாது

    தானாகச் சேமிப்பு. உலாவியை மூடினாலும் குறியீடு இருக்கும்.

    பிழை வந்தால் என்ன ஆகும்?

    ஆங்கிலப் பிழைச் செய்திகள் குழந்தைகளுக்குப் புரியும் வார்த்தைகளில் மொழிபெயர்க்கப்படும். சரி செய்வது எப்படி என்றும் காட்டும்.

    ஆங்கிலப் பிழை SyntaxError: '(' was never closed
    ↓
    மொழிபெயர்க்கப்பட்டது அடைப்புக்குறி ( மூடப்படவில்லை. ) சேர்த்து மூடு
    ஆங்கிலப் பிழை NameError: name 'hensuu' is not defined
    ↓
    மொழிபெயர்க்கப்பட்டது hensuu என்றால் என்ன? இன்னும் உருவாக்கவில்லை அல்லது பெயரைச் சரிபார்
    ஆங்கிலப் பிழை IndentationError: expected an indented block
    ↓
    மொழிபெயர்க்கப்பட்டது இடைவெளி போதாது. if அல்லது for பிறகு 4 இடைவெளி சேர்

    20-க்கும் மேற்பட்ட பிழை வகைகளை ஆதரிக்கிறது

    Scratch-க்குப் பிறகு என்ன?

    Scratch-இல் தொகுதிகளை இணைத்து நிரல் உருவாக்க கற்றுக்கொண்டாய். அருமை!

    ஆனால் எழுத்துக்களால் நிரல் எழுத விரும்பினால், அடுத்த படி Python.

    Python பயிற்சிப் புத்தகத்தில், Scratch-இல் கற்ற "திரும்பச்செய்" மற்றும் "என்றால்" — Python-இல் எழுதலாம்.

    செயலியின் "எடுத்துக்காட்டுகள்" பகுதியில் Scratch தொகுதிகள் போலவே செயல்படும் Python குறியீடு உள்ளது.

    எடுத்துக்காட்டுகளைப் பார்

    என்னவெல்லாம் செய்யலாம் பார்!
