    • 🌍 अनलाइन Python उपकरणहरू सबै अंग्रेजीमा — बालबालिकालाई गाह्रो

    🌐

    इन्स्टल गर्नुपर्दैन

    नेपालीमा

    बटन र मेनू नेपालीमा। सानासाना विद्यार्थीलाई सजिलो।

    बालबालिकालाई मिल्ने त्रुटि

    💾

    कोड हराउँदैन

    स्वत:-सेभ। ब्राउजर बन्द गरे पनि कोड त्यहीँ रहन्छ।

    त्रुटि आए के हुन्छ?

    अंग्रेजी त्रुटि सन्देश बालबालिकालाई बुझ्ने शब्दमा अनुवाद हुन्छन्। कसरी सच्याउने पनि बताउँछ।

    Scratch पछि के?

    Scratch मा ब्लकहरू जोडेर प्रोग्राम बनाउन सिक्यौ। शाबास!

    तर अक्षरले प्रोग्राम लेख्न चाहन्छौ भने, अर्को कदम Python हो।

    Python अभ्यास कापीमा, Scratch मा सिकेको "दोहोर्‍याउ" र "यदि" — Python मा लेख्न सकिन्छ।

    एपको "उदाहरणहरू" भागमा Scratch ब्लकहरू जस्तै काम गर्ने Python कोड छ।

    उदाहरणहरू हेर

    के के गर्न सकिन्छ हेर!
