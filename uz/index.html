    Xato boʻlsa nima boʻladi?

    Inglizcha xato xabarlari bolalar tushunadigan soʻzlarga tarjima qilinadi. Qanday tuzatishni ham koʻrsatadi.

    Inglizcha xato SyntaxError: '(' was never closed
    ↓
    Tarjima qilingan Qavs ( yopilmagan. ) qoʻshib yop
    Inglizcha xato NameError: name 'hensuu' is not defined
    ↓
    Tarjima qilingan hensuu nima? Hali yaratilmagan yoki imloni tekshir
    Inglizcha xato IndentationError: expected an indented block
    ↓
    Tarjima qilingan Boʻshliqlar yetarli emas. if yoki for dan keyin 4 ta boʻshliq qoʻy

    20 dan ortiq xato turlarini qoʻllab-quvvatlaydi

    Scratch dan keyin nima?

    Scratch da bloklarni birlashtirish orqali dastur yaratishni oʻrganding. Ajoyib!

    Lekin matn bilan dastur yozmoqchi boʻlsang, keyingi qadam Python.

    Python mashq daftarida, Scratch da oʻrgangan "takrorla" va "agar" — Python da yozish mumkin.

    Ilovaning "Namunalar" boʻlimida Scratch bloklari bilan bir xil ish bajaradigan Python kodi bor.
