# オンラインPython学習環境 - 要件定義

## 概要

ブラウザ上でPythonコードを入力・実行できるシンプルなWebアプリケーション。
サーバーサイド不要で、すべてクライアントサイドで完結する。

## 技術スタック

| 項目 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript（フレームワークなし） |
| コードエディタ | CodeMirror 6 |
| Python実行 | Pyodide（WebAssembly） |
| デプロイ先 | exe.dev VM（静的ファイル配信） |

## 機能要件

### F-01: コードエディタ

- CodeMirror 6 によるコード入力エリアを提供する
- Pythonのシンタックスハイライトを適用する
- 行番号を表示する

### F-02: コード実行

- 「実行」ボタンでエディタ内のPythonコードを実行する
- Pyodide（WebAssembly版Python）を使用し、ブラウザ内で実行する
- 実行中はローディング表示を行う

### F-03: 出力表示

- `print()` の出力を出力エリアに表示する
- エラー（SyntaxError, NameError等）も出力エリアに表示する
- 実行ごとに出力エリアをクリアする

### F-04: input() 対応

- `input()` 呼び出し時にブラウザの `prompt()` ダイアログで入力を受け付ける

### F-05: コード自動保存

- エディタの内容を `localStorage` に自動保存する
- ページ再読み込み時に保存済みコードを復元する

## 非機能要件

### NF-01: UI

- 日本語UI
- シンプル・ミニマルデザイン（白背景、機能重視）

### NF-02: 動作環境

- モダンブラウザ（Chrome, Firefox, Edge, Safari最新版）

### NF-03: パフォーマンス

- Pyodideの初回ロードに数秒かかることを許容する（ローディング表示で対応）
