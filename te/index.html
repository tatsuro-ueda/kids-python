    Scratch తర్వాత ఏమిటి?

    Scratch లో బ్లాక్‌లను కలిపి ప్రోగ్రామ్ చేయడం నేర్చుకున్నావు. అద్భుతం!

    కానీ టెక్స్ట్‌తో ప్రోగ్రామ్ రాయాలంటే, తదుపరి అడుగు Python.

    Python అభ్యాస పుస్తకంలో, Scratch లో నేర్చుకున్న "మళ్ళీ చేయి" మరియు "అయితే" — Python లో రాయవచ్చు.

    యాప్ లోని "ఉదాహరణలు" భాగంలో Scratch బ్లాక్‌ల లాగే పని చేసే Python కోడ్ ఉంది.
