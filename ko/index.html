    오류가 나면 어떡하지?

    영어 오류 메시지를 아이들이 이해할 수 있는 한국어로 바꿔줘요. 고치는 방법도 알려줘요.

    영어 오류 SyntaxError: '(' was never closed
    ↓
    번역 괄호 ( 가 닫히지 않았어. ) 를 넣어줘
    영어 오류 NameError: name 'hensuu' is not defined
    ↓
    번역 hensuu 이 뭐야? 아직 안 만들었거나 이름을 잘못 쓴 것 같아
    영어 오류 IndentationError: expected an indented block
    ↓
    번역 빈칸이 부족해. if 나 for 다음 줄에 빈칸 4개를 넣어줘

    20가지 이상의 오류 유형을 지원해요
