데이터 처리
Python 연습장의 모든 코드와 실행 결과는 완전히 브라우저 안에서 처리됩니다. 외부 서버로 데이터가 전송되지 않습니다.
로컬 저장소 사용
다음 용도로 브라우저의 로컬 저장소를 사용합니다:
  • 코드 자동 저장 (돌아왔을 때 그대로 있도록)
  • 즐겨찾기 알림을 봤는지 기억
  • 언어 설정 저장 (선택한 언어를 기억하도록)
쿠키
쿠키를 사용하지 않습니다.
분석
현재 분석 도구를 사용하지 않습니다. 미래에 추가하면 이 페이지를 업데이트하겠습니다.
