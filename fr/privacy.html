Politique de confidentialité

Traitement des données
Tout le code et les résultats dans le Cahier Python sont traités entièrement dans votre navigateur. Aucune donnée n'est envoyée à des serveurs externes.
Utilisation du stockage local
Nous utilisons le stockage local de votre navigateur aux fins suivantes :
  • Sauvegarder votre code automatiquement (pour qu'il soit là quand vous revenez)
  • Se souvenir si vous avez vu le rappel de favori
  • Sauvegarder votre préférence de langue (pour se souvenir de la langue choisie)
Cookies
Nous n'utilisons pas de cookies.
Analyses
Nous n'utilisons actuellement aucun outil d'analyse. Si nous en ajoutons à l'avenir, nous mettrons à jour cette page.
