# 要求内容

## 概要

LPに、CoderDojo活動者・石原淳也さんのツイート埋め込みを追加し、保護者の信頼判断を助ける。

## 背景

Passive Lookingフェーズの保護者は、プログラミング教育ツールの良し悪しを自分で判断する基準が少ない。権威ある教育者が実際の利用場面を語るツイートを掲載することで、「信頼できる人が認めているツール」という判断材料を提供する（marketing-problems.md P3対策）。

## 埋め込むツイート

- URL: https://x.com/jishiha/status/2038450310004556197
- 投稿者: 石原淳也（Junya Ishihara）@jishiha
- 投稿日: 2026-03-30
- 内容: CoderDojo調布で、Scratch経験のある小学生がPythonに挑戦した際に英語エラーメッセージで手こずり、日本語表示の要望が出たところ、その場でPythonれんしゅうちょうが使われた

## ユーザーストーリー

| ユーザー   | 保護者                | 小学生の子を持つ、プログラミング教育に関心のある保護者 |
| ------ | ------------------- | ----------------------------- |
| ジョブ    | 子どもに合うツールを選ぶ       | 子どもにPythonを学ばせたいが、どのツールが良いか迷っている |
| 課題     | 判断基準がない            | 自分がプログラミングに詳しくないため、ツールの良し悪しを評価できない |
| 従来のタスク | なんとなく有名なものを選ぶ     | 検索上位やScratch等の有名ツールに流れる、または諦める |
| 従来のコスト | 不安・ストレス           | 本当にこれでいいのか確信が持てないまま判断する |
| 新しいタスク | 教育者の声を読んで判断する     | LPに掲載された実際の利用エピソードを読み、安心して選択する |
| 新しいコスト | 数秒（ツイートを読む時間）    | 「教育の現場で使われている」という安心感を得られる |

## 受け入れ条件（Gherkin形式）

### ツイート埋め込みの表示

```gherkin
Given 保護者がLPを閲覧している
When  ページをスクロールして「保護者の方へ — 安心ポイント」セクションの直前まで到達する
Then  「教育の現場で使われています」という見出しとともに石原淳也さんのツイートが埋め込み表示されている
```

### ツイートの読み込み

```gherkin
Given LPが表示されている
When  Xの埋め込みスクリプトが読み込まれる
Then  ツイートがXの公式ウィジェット形式で表示される
```

### 埋め込みスクリプトが読み込めない場合

```gherkin
Given LPが表示されている
When  Xの埋め込みスクリプトがブロックまたはタイムアウトした
Then  ツイート本文がプレーンテキストのリンク付き引用として表示される（フォールバック）
```

### LPのパフォーマンスへの影響

```gherkin
Given LPが初回読み込みされる
When  ツイート埋め込みスクリプトが読み込まれる
Then  スクリプトはasyncで読み込まれ、LPの初期表示をブロックしない
```

## 成功指標

- LPに石原さんのツイートが表示される
- ツイートの読み込みがLPの初期表示を遅延させない
- 埋め込みスクリプトが読み込めない場合でも、引用テキストとリンクが表示される

## スコープ外

以下はこのフェーズでは実装しません:

- 複数のツイートや口コミの掲載
- 「ユーザーの声」セクションの新設
- 多言語LP版への展開
