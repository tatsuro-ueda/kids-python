# 課題: 保護者がツールの良し悪しを判断できない

## げんしょう

Passive Lookingフェーズの保護者は「子どもにPythonを学ばせたい」とぼんやり思っているが、
具体的にどのツールが良いかを判断する基準が少ない。

プログラミング教育ツールは数多くあり、保護者自身がプログラミングに詳しくない場合、
「このツールは本当に良いものなのか？」を自分で評価するのが難しい。

## なぜ困るか

判断基準がないと、保護者は以下の行動を取る:
- 検索上位のツールをなんとなく選ぶ
- 有名なブランド（Scratch等）に留まる
- 「よく分からないからやめておこう」と離脱する

結果として、Pythonれんしゅうちょうが保護者の選択肢に入らない。

## 対策: 権威ある人のツイートを掲載する

CoderDojo調布で活動する石原淳也さん（@jishiha）のツイートを引用掲載する。

> 普段はScratch、Python に挑戦したいという小学生の参加者が、英語のエラーメッセージで手こずっていて、
> 日本語で表示されるといいのにねえ、と言っていたら、CoderDojo 調布の時間中にできてしまいました。

このツイートが持つ価値:
- **実際の利用場面の描写**: 小学生がPythonに挑戦→英語エラーで困る→日本語化で解決
- **権威性**: CoderDojoという教育コミュニティの活動者による発言
- **ストーリー**: 保護者が自分の子どもに重ねて共感できる具体的なエピソード

marketing-problems.md の **P3（Passive Lookingフェーズの保護者・教育者にリーチできない）** への対策となる。
「信頼できる人が実際に使っている」という情報が、判断基準のない保護者に安心感を与える。
