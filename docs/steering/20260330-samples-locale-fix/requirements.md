# おてほんサンプルの言語切替不具合修正 — Requirements

## 概要

🌐で言語を切り替えた後、おてほんのサンプルコードが切替先の言語にならない不具合を修正する。

## 背景

UIは正しく切り替わる（例: スペイン語の「Ejecutar」「Ejemplos」）のに、サンプルコードは日本語のまま表示される。この製品の顧客課題は「英語ができない子どもがプログラミングを学べない」ことであり、サンプルコードが母語にならなければ課題が解決されない。

## 受け入れ条件

```gherkin
Given アプリを日本語で開いている
When  🌐でスペイン語に切り替え、おてほんからサンプルを選ぶ
Then  スペイン語のサンプルコードがエディタに挿入される
```

```gherkin
Given アプリをスペイン語で使っている
When  🌐で日本語に戻し、おてほんからサンプルを選ぶ
Then  日本語のサンプルコードがエディタに挿入される
```
