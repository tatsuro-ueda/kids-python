# requirements: input() のUI改善

## もんだい

- げんざいは `input()` が `prompt()` ダイアログで表示される
- ダイアログはブラウザのきのうで、こどもにはわかりにくい
- `input("ねんれいをすうじでにゅうりょくしてください: ")` のとき、プロンプトもじれつがダイアログに出るだけで、出力エリアとのつながりがない

## ゴール

- `input()` のプロンプトもじれつ（例: 「ねんれいをすうじでにゅうりょくしてください: 」）を出力エリアにインラインで表示する
- そのよこ（またはつづき）にテキスト入力欄を表示する
- ユーザーがにゅうりょくしてEnterをおすと、じっこうがつづく

## スコープ

- `prompt()` ダイアログをやめて、出力エリア内のインライン入力にかえる
- みためはこどもむけUIに合わせる（パステル、まるいかど）
