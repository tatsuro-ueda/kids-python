# コンテンツマーケティング・被リンク獲得 — Design

## 概要

外部プラットフォーム（note/Zenn）でのブログ記事公開とSNS発信を通じて、LPへの被リンクと検索流入を獲得する。個人開発者が最小コストで実行できる施策に絞る。

---

## 1. コンテンツ制作フロー

```
テーマ選定 → 記事執筆 → プロダクトCTA挿入 → 投稿 → SNSで告知 → 効果計測
                                                              ↓
                                                    seo-tracking.mdに記録
```

---

## 2. 記事テンプレート

### 2.1 ハウツー記事の構成

```markdown
# [タイトル: ターゲットKWを含む]

## はじめに
[読者の困りごとに共感 → この記事で解決できることを提示]

## [本題: 3-5セクション]
[具体的な手順・情報]

## まとめ
[要点の振り返り]

## Pythonれんしゅうちょうで試してみよう
[プロダクトの紹介 + リンク]
「ブラウザだけでPythonが動く。インストール不要、ぜんぶひらがな。」
→ [Pythonれんしゅうちょう](https://[DOMAIN]/)
```

### 2.2 教育リソース記事の構成

```markdown
# [タイトル: 「早見表」「一覧」「チートシート」等を含む]

## 概要
[このリソースの使い方]

## [リソース本体: 表・リスト・図解]
[そのまま使える実用的な内容]

## ダウンロード
[PDF版のリンク（あれば）]

## 関連ツール
[プロダクトの紹介 + リンク]
```

## 3. 初期コンテンツの詳細設計

### C1: 小学生がPythonを始めるための完全ガイド

- **投稿先**: note
- **ターゲットKW**: Python 小学生 始め方
- **想定文字数**: 3,000-4,000字
- **構成**:
  1. なぜ小学生にPythonなのか（Scratch→Pythonの流れ）
  2. Python学習に必要な準備（→ 「準備不要」のPythonれんしゅうちょうを紹介）
  3. 最初に書くコード3つ（print, 計算, 変数）
  4. つまずきポイントと解決法
  5. まとめ + CTA

### C2: ChromebookでPythonを使う3つの方法

- **投稿先**: Zenn（技術系読者向け）
- **ターゲットKW**: Chromebook Python
- **想定文字数**: 2,000-3,000字
- **構成**:
  1. ChromebookでPythonを使う需要（GIGAスクール）
  2. 方法1: Linux開発環境（上級者向け）
  3. 方法2: オンラインIDE（Replit等）
  4. 方法3: Pythonれんしゅうちょう（子ども向けに最適）
  5. 比較表 + まとめ

### C4: 子どもが書いたPythonコード10選

- **投稿先**: note
- **ターゲットKW**: Python サンプルコード 子ども
- **想定文字数**: 2,000-2,500字
- **構成**:
  1. はじめに（コピペで動くよ！）
  2. コード10選（各コード + 実行結果のスクリーンショット）
  3. 「Pythonれんしゅうちょうでそのまま動かせます → URL」

## 4. 被リンク獲得のアプローチ

### 4.1 教育系まとめ記事への掲載

**手順:**

1. 「子ども プログラミング ツール まとめ」等で検索
2. まとめ記事の運営者・著者を特定
3. 以下のテンプレートで連絡

```
件名: 子ども向けPythonツールのご紹介

[名前]様

突然のご連絡失礼いたします。
「Pythonれんしゅうちょう」という子ども向けPython学習環境を開発している[名前]と申します。

[記事タイトル]を拝見し、大変参考になりました。

もしよろしければ、弊ツール「Pythonれんしゅうちょう」もご紹介いただけますと幸いです。
特徴: インストール不要、ひらがなUI、日本語エラー、完全無料
URL: https://[DOMAIN]/

ご検討いただけますと嬉しいです。
```

### 4.2 コミュニティへの参加

| コミュニティ | アクション |
|---|---|
| Scratch日本語フォーラム | Scratch→Python移行の質問に回答 |
| teratail | 子ども向けPython環境の質問に回答 |
| X #プログラミング教育 | ハッシュタグで定期的に発信 |
| Qiita | 技術記事を投稿（Pyodide/WASM系） |

## 5. SNS運用設計

### 5.1 Xアカウント運用

- **アカウント名**: Pythonれんしゅうちょう（仮）
- **プロフィール**: プロダクトの一文説明 + URL
- **投稿スケジュール**:
  - 月: コード例紹介
  - 水: 教育Tips or アップデート
  - 金: コード例紹介
- **ハッシュタグ**: #プログラミング教育 #Python #小学生 #Scratch卒業

### 5.2 投稿テンプレート

**コード例紹介:**
```
🐍 今日のPythonコード

[コード例のスクリーンショット or テキスト]

→ Pythonれんしゅうちょうでそのまま動かせるよ！
https://[DOMAIN]/

#プログラミング教育 #Python
```

## 6. 効果計測

### 6.1 記事ごとの計測

| 指標 | 計測方法 |
|---|---|
| PV数 | note/Zennの管理画面 |
| いいね/スキ数 | note/Zennの管理画面 |
| LP流入数 | GSCのリファラーデータ |
| 被リンク | GSCの「リンク」レポート |

### 6.2 記録先

`docs/seo-tracking.md`にコンテンツマーケティングセクションを追加:

```markdown
### コンテンツマーケティング

| 記事 | 投稿日 | PV | スキ | LP流入 | 被リンク |
|---|---|---|---|---|---|
| C1: Python始め方ガイド | - | - | - | - | - |
```
