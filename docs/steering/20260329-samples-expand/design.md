# おてほん拡充 — 設計

## 方針

- 全11種類（既存5つを再編 + 新規6つ）
- 前半(1-5)は3〜5行、後半(6-11)は長めでもOK（10行前後まで）
- 各サンプルの1行目にScratch対応コメント
- 変数名はひらがな日本語
- おみくじは独立サンプルとして残す

## 変更方針

`src/samples.js` の `samples` 配列を11要素に差し替え。UIやロジック変更は不要（ドロップダウンは配列から自動生成されている想定）。

## サンプルコード一覧

### 1. はじめまして（既存・変更なし）

```python
# Scratch: 「こんにちは！という」ブロックとおなじだよ
print("こんにちは！")
```

**学ぶこと:** `print()`

### 2. なまえをきいてみよう（既存・変更なし）

```python
# Scratch: 「○○とこたえて待つ」ブロックとおなじだよ
なまえ = input("なまえをおしえて: ")
print(なまえ + "さん、こんにちは！")
```

**学ぶこと:** `input()`、変数、文字列結合

### 3. たしざんけいさんき（新規）

```python
# Scratch: 「○ + ○」えんざんブロックとおなじだよ
あ = int(input("ひとつめのすうじ: "))
い = int(input("ふたつめのすうじ: "))
print("たしざん: " + str(あ + い))
print("ひきざん: " + str(あ - い))
print("かけざん: " + str(あ * い))
```

**学ぶこと:** `int()`、算術演算、`str()`

### 4. もしも...なら（既存・変更なし）

```python
# Scratch: 「もし○なら」ブロックとおなじだよ
てんき = input("きょうのてんきは？(はれ/あめ): ")
if てんき == "はれ":
    print("おそとであそぼう！")

    print("おうちでプログラミングしよう！")
```

**学ぶこと:** `if/else`、`==`比較

### 5. いろいろあいさつ（新規）

```python
# Scratch: 「もし○なら、でなければもし○なら」ブロックとおなじだよ
じかん = int(input("いまなんじ？(0〜23): "))
if じかん < 10:
    print("おはよう！")

    print("こんにちは！")
```

**学ぶこと:** `if/elif/else`、`<`比較、`int()`

### 6. くりかえそう（既存・変更なし）

```python
# Scratch: 「○かいくりかえす」ブロックとおなじだよ
for かいすう in range(5):
    print(str(かいすう + 1) + "かいめ: こんにちは！")
```

**学ぶこと:** `for`、`range()`

### 7. カウントダウン（新規）

```python
# Scratch: 「○ずつかえる」+「○までくりかえす」ブロックとおなじだよ
かず = 10
while かず > 0:
    print(かず)
    かず = かず - 1
print("はっしゃ！🚀")
```

**学ぶこと:** `while`ループ、変数の更新

### 8. なんばんめ？（新規）

```python
# Scratch: リストの「○ばんめ」ブロックとおなじだよ
くだもの = ["りんご", "みかん", "ぶどう", "いちご", "もも"]
print("くだものリスト: " + str(くだもの))
ばんごう = int(input("なんばんめをみる？(1〜5): "))
print(str(ばんごう) + "ばんめは " + くだもの[ばんごう - 1] + " だよ！")
```

**学ぶこと:** リスト、インデックス（Scratchは1始まり→Pythonは0始まりの違い）

### 9. もじをかぞえよう（新規）

```python
# Scratch: 「○のながさ」ブロックとおなじだよ
ことば = input("すきなことばをいれてね: ")
print("「" + ことば + "」は " + str(len(ことば)) + "もじだよ！")
for もじ in ことば:
    print("→ " + もじ)
```

**学ぶこと:** `len()`、文字列のforループ

### 10. おみくじ（既存・変更なし）

```python
# Scratch: 「○から○までのらんすう」ブロックとおなじだよ
import random
けっか = random.choice(["だいきち", "ちゅうきち", "しょうきち", "きち", "すえきち"])
print("きょうのうんせいは..." + けっか + "！")
```

**学ぶこと:** `import`、`random.choice()`

### 11. すうじあてゲーム（新規）

```python
# いろんなブロックをくみあわせたゲームだよ！
import random
こたえ = random.randint(1, 10)
print("1から10のすうじをあてよう！")

かいすう = 0
while True:
    よそう = int(input("すうじをいれて: "))
    かいすう = かいすう + 1
    if よそう == こたえ:
        print("あたり！🎉 " + str(かいすう) + "かいでせいかい！")
        break
    elif よそう < こたえ:
        print("もっとおおきいよ")
    else:
        print("もっとちいさいよ")
```

**学ぶこと:** 総合（`while True`、`break`、`random`、`if/elif/else`、`input`）
