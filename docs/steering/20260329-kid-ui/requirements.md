# requirements: こどもむけUI

## たいしょうユーザー

- しょうがっこう1〜2ねんせい（ていがくねん）

## デザインほうしん

### いろあい
- パステルカラー（やわらかいピンク、みずいろ、きいろなど）

### はいけい
- もようつき（ドット、ストライプなど）

### フォント
- ふつうのゴシックたい（とくべつなフォントは使わない）

### ボタン
- まるっこいパステルボタン

### キャラクター
- ヘビ（Python = ニシキヘビ）のキャラクター
- ヘッダーにいるだけ（リアクションはしない）
- フリー素材を使う

## スコープ

- みためだけのへんこう
- きのう（エラーメッセージ、エディタ、じっこうなど）はさわらない

## へんこうたいしょう

- `style.css` — いろ・はいけい・ボタンスタイルの変更
- `index.html` — キャラクター画像の追加、必要に応じた構造変更
- `dist/bundle.js` — 再ビルド（JSに変更がある場合のみ）
