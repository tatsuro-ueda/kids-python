## 計測ルール

- **計測頻度**: 毎週水曜日
- **手動検索**: Googleシークレットモード、PC（+余裕があればモバイル）
- **GSC**: 過去7日間のデータを記録（水曜計測 → 前週水曜〜火曜のデータ）
- **順位**: 上位50位以内のみ記録。圏外は `-`
- **最新の記録を上に配置**（逆時系列）
- **「気づき・次のアクション」**: 必ず「事実 → アクション」形式で記述

## 週次計測チェックリスト（毎週水曜に実施）

1. GSC → 検索パフォーマンス → 過去7日間 → クエリタブを確認
2. ターゲットKW（KW1-KW8）の表示・クリック・CTR・順位を記録
3. 想定外クエリ（ターゲットKW以外で表示・クリックがあるもの）を確認・記録
4. カバレッジレポートでエラーがないか確認
5. 「気づき・次のアクション」を記入
6. 隔週（第1・第3水曜）: 直近2週分のデータを比較し、改善アクションを実施
   - 表示 > 100 かつ CTR < 2% → title/meta description 改善
   - 順位 30位以下 → LP本文のテキスト強化
   - 想定外KWで表示 > 50 → コンテンツ追加を検討
   - 前週比で順位5以上下落 → 競合分析
