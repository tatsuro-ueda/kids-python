    Pernah mengalami ini?

    • 😢 Mencoba menginstal Python, tapi terlalu sulit dan menyerah

    • 🌍 Alat Python online semuanya dalam bahasa Inggris — terlalu sulit untuk anak-anak
    • ❓ Pesan error dalam bahasa Inggris — tidak tahu apa yang harus diperbaiki

    Fitur

    🌐

    Tanpa instalasi

    Cukup buka browser. Bisa di Chromebook dan tablet juga.

    ✏️

    Dalam bahasa Indonesia

    Tombol dan menu dalam bahasa Indonesia. Mudah untuk anak-anak.

    💬

    Error ramah anak

    Error bahasa Inggris diterjemahkan ke kata-kata yang anak-anak pahami. Juga menunjukkan cara memperbaikinya.

    💾

    Kode tidak pernah hilang

    Simpan otomatis. Tutup browser dan kodemu masih ada.

    Kalau ada error?

    Pesan error bahasa Inggris diterjemahkan ke kata-kata yang anak-anak pahami. Juga menunjukkan cara memperbaikinya.

    Error bahasa Inggris SyntaxError: '(' was never closed
    ↓
    Diterjemahkan Tanda kurung ( belum ditutup. Tambahkan ) untuk menutupnya
    Error bahasa Inggris NameError: name 'hensuu' is not defined
    ↓
    Diterjemahkan Apa itu hensuu? Belum dibuat, atau periksa namanya
    Error bahasa Inggris IndentationError: expected an indented block
    ↓
    Diterjemahkan Spasi kurang. Tambahkan 4 spasi setelah if atau for

    Mendukung lebih dari 20 jenis error

    Setelah Scratch, apa selanjutnya?

    Kamu sudah belajar membuat program dengan menyusun blok di Scratch. Hebat!

    Tapi kalau ingin menulis program dengan teks, langkah berikutnya adalah Python.

    Dengan Buku Latihan Python, kamu bisa menulis blok "ulangi" dan "jika" yang sama seperti di Scratch — tapi dalam Python.

    Bagian "Contoh" di aplikasi punya kode Python yang melakukan hal yang sama seperti blok Scratch.
