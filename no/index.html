    Hva om det blir feil?

    Engelske feilmeldinger oversettes til ord som barn forstar. Viser ogsa hvordan du fikser dem.

    Engelsk feil SyntaxError: '(' was never closed
    ↓
    Oversatt Parentesen ( ble aldri lukket. Legg til ) for a lukke den
    Engelsk feil NameError: name 'hensuu' is not defined
    ↓
    Oversatt Hva er hensuu? Du har ikke laget den enna, eller sjekk stavingen
    Engelsk feil IndentationError: expected an indented block
    ↓
    Oversatt For fa mellomrom. Legg til 4 mellomrom etter if eller for

    Hva kommer etter Scratch?

    Du laerte a lage programmer ved a sette sammen blokker i Scratch. Kult!

    Men hvis du vil skrive programmer med tekst, er neste steg Python.

    Med Python-skrivebok kan du skrive de samme "gjenta" og "hvis" blokkene fra Scratch — men i Python.

    Appens "Eksempler" har Python-kode som gjor det samme som Scratch-blokker.

    Se hva du kan gjore!
