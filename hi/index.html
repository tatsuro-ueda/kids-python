Python अभ्यास पुस्तिका

ब्राउज़र में ही Python चलाओ।
बच्चों के लिए प्रोग्रामिंग अभ्यास पुस्तिका

इंस्टॉल नहीं चाहिए। हिंदी में एरर। समझने में आसान।

अभी आज़माओ

क्या ऐसा हुआ है?

  • 😢 Python इंस्टॉल करने की कोशिश की, लेकिन बहुत मुश्किल लगा
  • 💻 Chromebook पर सॉफ़्टवेयर इंस्टॉल नहीं कर सकते
  • 🌍 ऑनलाइन Python टूल सब अंग्रेज़ी में हैं — बच्चों के लिए मुश्किल
  • ❓ एरर मैसेज अंग्रेज़ी में — समझ नहीं आता क्या ठीक करें

खासियतें

🌐

इंस्टॉल नहीं चाहिए

बस ब्राउज़र खोलो। Chromebook और टैबलेट पर भी चलता है।

हिंदी में

बटन और मेनू हिंदी में। छोटे बच्चों के लिए आसान।

💬

बच्चों के लिए एरर

अंग्रेज़ी एरर को बच्चों की भाषा में बदलता है। ठीक करने का तरीका भी बताता है।

💾

कोड कभी नहीं खोता

ऑटो-सेव। ब्राउज़र बंद करो फिर भी कोड वहीं रहता है।

अगर एरर आ जाए तो?

अंग्रेज़ी एरर मैसेज को बच्चों की समझ में आने वाली हिंदी में बदलता है। ठीक करने का तरीका भी बताता है।

Scratch के बाद क्या?

Scratch में ब्लॉक जोड़कर प्रोग्राम बनाना सीख लिया। शाबाश!

अगर अब टेक्स्ट में प्रोग्राम लिखना चाहते हो, तो अगला कदम Python है।

Python अभ्यास पुस्तिका में Scratch की "दोहराओ" और "अगर" को Python में लिख सकते हो।

ऐप के "उदाहरण" में Scratch ब्लॉक जैसा करने वाला Python कोड है।

उदाहरण देखो

देखो क्या-क्या कर सकते हो!

🎨 पैटर्न बनाओ

for i in range(5):
    print("★" * i)
