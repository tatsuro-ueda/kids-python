    Daca apare o eroare?

    Mesajele de eroare in engleza sunt traduse in cuvinte pe care copiii le inteleg. Arata si cum sa le repari.

    Eroare in engleza SyntaxError: '(' was never closed
    ↓
    Tradus Paranteza ( nu a fost inchisa. Adauga ) ca sa o inchizi
    Eroare in engleza NameError: name 'hensuu' is not defined
    ↓
    Tradus Ce este hensuu? Nu l-ai creat inca, sau verifica ortografia
    Eroare in engleza IndentationError: expected an indented block
    ↓
    Tradus Nu sunt destule spatii. Adauga 4 spatii dupa if sau for

    Suporta peste 20 de tipuri de erori diferite

    Ce vine dupa Scratch?

    Ai invatat sa faci programe punand blocuri impreuna in Scratch. Minunat!

    Dar daca vrei sa scrii programe cu text, urmatorul pas este Python.

    Cu Caietul Python poti scrie aceleasi blocuri "repeta" si "daca" din Scratch — dar in Python.

    Sectiunea "Exemple" are cod Python care face aceleasi lucruri ca blocurile Scratch.
