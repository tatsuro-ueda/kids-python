E se der um erro?

Mensagens de erro em inglês são traduzidas para palavras que crianças entendem. Também mostra como corrigir.

Erro em inglês SyntaxError: '(' was never closed
↓
Traduzido O parêntese ( não foi fechado. Coloque ) para fechar
Erro em inglês NameError: name 'hensuu' is not defined
↓
Traduzido O que é hensuu? Você ainda não criou, ou confira o nome
Erro em inglês IndentationError: expected an indented block
↓
Traduzido Faltam espaços. Coloque 4 espaços depois de if ou for

O que vem depois do Scratch?

Você aprendeu a fazer programas juntando blocos no Scratch. Incrível!

Mas se quiser escrever programas com texto, o próximo passo é Python.

Com o Caderno de Python, você pode escrever os mesmos blocos de "repetir" e "se" do Scratch — mas em Python.

A seção "Exemplos" tem código Python que faz as mesmas coisas que os blocos do Scratch.
