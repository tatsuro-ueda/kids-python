Política de privacidade

Tratamento de dados
Todo o código e resultados no Caderno de Python são processados inteiramente no seu navegador. Nenhum dado é enviado para servidores externos.
Uso do armazenamento local
Usamos o armazenamento local do seu navegador para os seguintes fins:
  • Salvar seu código automaticamente (para que esteja lá quando você voltar)
  • Lembrar se você já viu o lembrete de favoritos
  • Salvar sua preferência de idioma (para lembrar o idioma escolhido)
Cookies
Não usamos cookies.
Análises
Atualmente não usamos ferramentas de análise. Se adicionarmos no futuro, atualizaremos esta página.
