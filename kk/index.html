Scratch-тан кейін не?

Scratch-та блоктарды қосып бағдарлама жасауды үйрендің. Керемет!

Бірақ мәтінмен бағдарлама жазғың келсе, келесі қадам — Python.

Python жаттығу дәптерінде Scratch-та үйренген "қайтала" мен "егер" — Python-да жазуға болады.

Қолданбаның "Үлгілер" бөлімінде Scratch блоктары сияқты жұмыс істейтін Python коды бар.

Үлгілерді көр
