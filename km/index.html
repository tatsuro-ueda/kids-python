    សាកឥឡូវ

    ស្គាល់ទេ?

    • 😢 សាកដំឡើង Python តែវាពិបាកពេក បានបោះបង់
    • 💻 Chromebook ដំឡើងកម្មវិធីមិនបានទេ
    • 🌍 ឧបករណ៍ Python អនឡាញសុទ្ធតែជាភាសាអង់គ្លេស — ពិបាកសម្រាប់កុមារ
    • ❓ សារកំហុសជាភាសាអង់គ្លេស — មិនដឹងត្រូវកែអ្វីទេ

    🌐

    មិនចាំបាច់ដំឡើង

    បើកកម្មវិធីរុករកតែម្តង។ ដំណើរការលើ Chromebook និង tablet ដែរ។

    ✏️

    ជាភាសាខ្មែរ

    ប៊ូតុងនិងម៉ឺនុយជាភាសាខ្មែរ។ ងាយស្រួលសម្រាប់សិស្សតូចៗ។

    💬

    កំហុសដែលកុមារយល់បាន

    កំហុសអង់គ្លេសត្រូវបានបកប្រែជាពាក្យដែលកុមារយល់បាន។ បង្ហាញរបៀបកែផងដែរ។

    💾

    កូដមិនបាត់ទេ

    រក្សាទុកដោយស្វ័យប្រវត្តិ។ បិទកម្មវិធីរុករកហើយកូដនៅតែមាន។

    បើមានកំហុសវិញ?

    សារកំហុសអង់គ្លេសត្រូវបានបកប្រែជាពាក្យដែលកុមារយល់បាន។ បង្ហាញរបៀបកែផងដែរ។

    កំហុសអង់គ្លេស SyntaxError: '(' was never closed
    ↓
    បកប្រែ វង់ក្រចក ( មិនទាន់បិទ។ បន្ថែម ) ដើម្បីបិទវា
    កំហុសអង់គ្លេស NameError: name 'hensuu' is not defined
    ↓
    បកប្រែ hensuu ជាអ្វី? អ្នកមិនទាន់បង្កើតវាទេ ឬពិនិត្យអក្ខរាវិរុទ្ធ
    កំហុសអង់គ្លេស IndentationError: expected an indented block
    ↓
    បកប្រែ ដកឃ្លាមិនគ្រប់។ បន្ថែមដកឃ្លា 4 បន្ទាប់ពី if ឬ for

    គាំទ្រប្រភេទកំហុសជាង 20

    បន្ទាប់ពី Scratch ជាអ្វី?

    អ្នកបានរៀនបង្កើតកម្មវិធីដោយដាក់ប្លុកក្នុង Scratch។ ពូកែ!

    ប៉ុន្តែបើអ្នកចង់សរសេរកម្មវិធីដោយអក្សរ ជំហានបន្ទាប់គឺ Python។

    នៅក្នុងសៀវភៅហាត់ Python អ្នកអាចសរសេរ "ធ្វើម្ដងទៀត" និង "បើ" ពី Scratch — ជា Python វិញ។

    ផ្នែក "ឧទាហរណ៍" នៃកម្មវិធីមានកូដ Python ដែលធ្វើដូចប្លុក Scratch។
