Python පුහුණු පොත

පෞද්ගලිකත්ව ප්‍රතිපත්තිය

Data හැසිරවීම
Python පුහුණු පොතේ code සහ ප්‍රතිඵල සම්පූර්ණයෙන්ම ඔබේ browser එකේම process වෙනවා. බාහිර servers වලට කිසිම data යවන්නේ නැහැ.
Local Storage භාවිතය
පහත අරමුණු සඳහා ඔබේ browser එකේ local storage භාවිතා කරනවා:
  • ඔබේ code auto-save කිරීම (ආපහු ආවම code එක තියෙන්න)
  • Bookmark reminder පෙන්නුවාද කියලා මතක තබා ගැනීම
  • ඔබේ භාෂා preference save කිරීම (තෝරපු භාෂාව මතක තබා ගන්න)
Cookies
අපි cookies භාවිතා කරන්නේ නැහැ.
Analytics
දැනට analytics tools කිසිවක් භාවිතා කරන්නේ නැහැ. අනාගතයේ එකතු කළොත් මේ පිටුව update කරනවා.
