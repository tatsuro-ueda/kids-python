    Scratch नंतर काय?

    Scratch मध्ये ब्लॉक्स जोडून प्रोग्रॅम बनवायला शिकलास. शाबास!

    पण मजकुरात प्रोग्रॅम लिहायचा असेल तर पुढचा टप्पा Python आहे.

    Python सराव वहीत, Scratch मधल्या "पुन्हा करा" आणि "जर" — Python मध्ये लिहिता येतात.

    अॅपच्या "उदाहरणे" विभागात Scratch ब्लॉक्ससारखेच काम करणारा Python कोड आहे.

    उदाहरणे बघा
