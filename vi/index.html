Sau Scratch thì học gì?

Bạn đã biết ghép các khối để tạo chương trình trong Scratch rồi. Giỏi lắm!

Nhưng nếu muốn viết chương trình bằng chữ, bước tiếp theo là Python.

Với Sổ tay Python, bạn có thể viết các khối "lặp" và "nếu" giống Scratch — nhưng bằng Python.

Phần "Ví dụ" trong ứng dụng có code Python làm những thứ giống như khối Scratch.
