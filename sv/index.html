Vad kommer efter Scratch?

Du larde dig gora program genom att satta ihop block i Scratch. Fantastiskt!

Men om du vill skriva program med text ar nasta steg Python.

Med Python-hafte kan du skriva samma "upprepa" och "om" block fran Scratch — men i Python.

Appens "Exempel" har Python-kod som gor samma saker som Scratch-block.
